Installation Instructions

Installing P&T Field Pack on ExpressionEngine 2.x

Prerequisites

Make sure your system meets the minimum requirements:

  • ExpressionEngine 2.1.3 or later
  • PHP 5

Installation Instructions

  1. Upload all of the folders within ee2/third_party/ to system/expressionengine/third_party/
  2. Enable whichever fieldtypes you wish to use in Add-Ons → Fieldtypes

Adding Fields to the Publish Page

Once P&T Field Pack is installed, you can add its fieldtypes to the Publish Page by following these steps:

  1. Go to Admin → Channel Administration → Custom Fields
  2. Click the “Add/Edit Custom Fields” link beside your field group
  3. Click the “Create a New Custom Field” button
  4. Give the new field a name and a label, and choose the appropriate fieldtype in the Field Type dropdown.
  5. Click the Submit button.

That’s how you create new Publish Page fields in ExpressionEngine.
